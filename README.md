# web_socket.dart

Dart Abstraction on WebSocket

## Usage

### Common

pubspec.yaml

````yaml
dependencies
  tekartik_web_socket:
    git:
      url: git://github.com/tekartik/web_socket.dart
      path: web_socket
      ref: null_safety
    version: '>=0.1.0'
````
